[AI] scheduleCalendar 날짜 색변경

개요
scheduleCalendar 컴포넌트에서 특정 날짜의 배경색을 변경하고 싶습니다.
예를 들어 공휴일이나 특정 이벤트가 있는 날짜를 다른 색상으로 표시하려고 합니다.
재현 방법
1. scheduleCalendar 컴포넌트 생성
2. 특정 날짜 (예: 2026-03-27) 배경색을 빨간색으로 변경 시도
3. setDateStyle API 사용 시 적용되지 않는 현상 발생

안녕하세요. scheduleCalendar 컴포넌트의 날짜 배경색 변경 방법을 안내드립니다.

1. setDateCellStyle API 사용

scheduleCalendar에서는 setDateStyle 대신 setDateCellStyle API를 사용해야 합니다.

// 특정 날짜 배경색 변경
var calendar = $p.getComponentById("scheduleCalendar1");
calendar.setDateCellStyle("2026-03-27", {
    "background-color": "#fee2e2",
    "color": "#dc2626"
});

// 여러 날짜 일괄 적용
var holidays = ["2026-03-01", "2026-05-05", "2026-06-06"];
holidays.forEach(function(date) {
    calendar.setDateCellStyle(date, {
        "background-color": "#fee2e2"
    });
});

2. CSS 클래스 활용 방법

스타일시트에 클래스를 정의하고 setDateCellClass를 사용할 수도 있습니다.

/* CSS */
.holiday-cell {
    background-color: #fee2e2 !important;
    color: #dc2626 !important;
}

// JavaScript
calendar.setDateCellClass("2026-03-27", "holiday-cell");

참고사항

- SP4 버전 이상에서 지원됩니다.
- 동적으로 변경 시 refresh() 호출이 필요할 수 있습니다.
