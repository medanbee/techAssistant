W-Tech 게시판 기술문의 AI 자동화

W-Tech QNA 게시판에 AI 자동 답변 기능을 도입하여 기술지원 업무를 자동화하는 프로젝트

1 프론트엔드 (W-Tech 게시판 UI)

W-Tech 게시판 UI에 AI 답변 및 사용자 인터랙션 기능 추가

기능 추가 리스트

  • Q&A 게시글 등록 페이지 개선
    • 제목, 웹스퀘어 버전, 회사명/프로젝트 입력 필드 추가
    • 유지보수 계약 여부 확인 안내 문구 추가 (계약되어 있어야 답변 진행 가능)
  • 회사명/프로젝트 검색 기능
    • 기술지원 시스템 DB에서 유지보수 계약 확인 (지원 날짜 기준)
    • [기타] 카테고리 선택 시 AI 자동 답변 대상에서 제외
  • 게시글 형식 수정
  • AI 답변 영역 UI 추가
    • 답변 내용 표시 (답변 신뢰도 함께 표시)
    • "엔지니어 답변 추가 요청" 버튼
    • 문의자 답글에 대한 AI 재답변 처리 (방식 검토 필요)
    • 새로 등록되는 글부터 적용 (기존 글 처리 방안 검토 필요)
  • 라이선스 발급
    • 라이선스 발급 UI 추가 → 데모 라이선스 자동 발급 기능

2 서버/백엔드 (API 서버)

AI 답변 생성 및 W-Tech와의 연동을 처리하는 API 서버 구성

처리 흐름

W-Tech 게시글 등록 → API 서버 (TechAssistant) → RAG 벡터 검색 (ChromaDB + multilingual-e5-base) → Claude API 호출 (답변 생성) → API 검증 (답변 내 API/이벤트/속성명 존재 확인) → 검수 큐 적재 → 담당자 승인 → W-Tech에 답변 등록

API 구현 현황

엔드포인트기능상태
POST /api/scanW-Tech 신규 미답변 문의 감지 + 자동 답변 생성 (SSE)완료
GET /api/queue검수 큐 조회 (pending / approved / rejected)완료
PATCH /api/queue/:id답변 수정 및 상태 변경 (승인/반려)완료
POST /api/answer단건 답변 생성 (인증 필요)완료
POST /api/searchRAG 유사 사례 검색완료
GET /api/health헬스 체크완료
W-Tech 답변 등록 API승인된 답변을 W-Tech 게시판에 등록미구현
